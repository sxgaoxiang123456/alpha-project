# 调研主题 2：A 股行情数据来源

> 调研日期：2026-05-25
> 调研方法：WebSearch + WebFetch（Claude Code 内置）与 muyu-search-mcp 双路交叉验证

---

## 一、全市场数据源盘点

A 股行情数据的获取渠道可分为 **6 大类**：

| 类别 | 代表渠道 | 核心特点 |
|------|---------|---------|
| **开源 Python 库** | AKShare、Baostock、Ashare | 免费、即装即用、无注册门槛 |
| **积分/付费 API** | Tushare Pro、iTick、聚合数据 | 标准化接口、质量可控、成本适中 |
| **机构级终端** | Wind（万得）、iFinD（同花顺） | 数据最全、质量最高、价格昂贵 |
| **券商量化接口** | QMT、PTrade | 直连交易所、Tick 级、需开户 |
| **量化云平台** | 聚宽（JoinQuant）、掘金（MyQuant） | 数据+回测一体、免费额度充足 |
| **交易所官方** | 上交所、深交所信息公司 | 最权威、仅面向机构/供应商 |
| **爬虫（高风险）** | 东方财富、雪球、新浪财经 | 免费、法律风险高、稳定性差 |

## 二、各渠道事实对照详表

### 2.1 开源 Python 库

#### AKShare

| 维度 | 事实 |
|------|------|
| **覆盖范围** | A 股、港股、美股、期货、外汇、基金、债券、宏观等 |
| **频率** | 日线 / 分钟线（仅当天或近 2 月）/ Tick（近 1 月） |
| **实时性** | 依赖爬虫源，比 Tushare 延迟 **15-30 分钟** |
| **限频与配额** | 无官方限频，但数据源（东方财富）限流约 **3 次/秒/IP** |
| **价格档位** | **完全免费** |
| **注册门槛** | 无需注册，pip 安装即用 |
| **商用合规** | 无商用授权，底层为爬虫，**商用法律风险高** |
| **稳定性** | 差。2025 年东方财富反爬大幅升级，频繁出现 IP 封禁、验证码、接口失效 [[1]](https://cloud.tencent.com/developer/article/2671369) |

#### Baostock

| 维度 | 事实 |
|------|------|
| **覆盖范围** | 专注 A 股，含历史 K 线、财务数据、指数成分 |
| **频率** | 日/周/月/分钟线 |
| **实时性** | 非实时，历史数据为主 |
| **限频与配额** | 无明确公开限制，适合个人/小批量调用 |
| **价格档位** | **完全免费** |
| **注册门槛** | 无需注册，匿名账号即可调用 |
| **商用合规** | 官方定位学术/研究用途，未明确商用授权 |
| **稳定性** | **优于 AKShare**。自有 API 通道，不受东方财富反爬直接影响 [[2]](https://www.baostock.com/) |

### 2.7 爬虫（高风险 ⚠️）

| 维度 | 东方财富 | 雪球 |
|------|---------|------|
| **覆盖范围** | A 股全市场实时行情、财务、研报 | A 股行情 + 社区讨论 |
| **频率** | 网页级（非 API） | 网页级（AJAX 动态加载） |
| **实时性** | 接近实时 | 接近实时 |
| **限频与配额** | 反爬极严，2025 年升级后约 **3 次/秒/IP** 即可能触发封禁 |
| **价格档位** | 免费 | 免费 |
| **注册门槛** | 无 | 无 |
| **商用合规** | **高风险**。《反不正当竞争法》第 12 条、网络数据安全管理条例第 18 条均对爬虫有严格限制。绕过反爬措施可能构成"非法获取计算机信息系统数据罪" [[11]](https://www.spp.gov.cn/llyj/202202/t20220210_543998.shtml) |
| **稳定性** | 差，接口随时变更，需持续维护 |

> **法律风险总结**：个人学习、低频、非商业爬虫处于灰色地带；任何**商业用途、高频调用、绕过技术防护措施**的爬虫行为均面临被诉不正当竞争甚至刑事责任的风险。

## 三、个人开发者主流选择

### 实际调研结论

根据中国量化社区（CSDN、知乎、掘金社区、叩富网）的开发者实际使用反馈，**个人开发者最常用的组合是：**

#### 第一梯队（最常用）

| 组合 | 使用场景 | 原因 |
|------|---------|------|
| **AKShare + Tushare 免费版** | 策略研究、回测、学习 | 零成本、Python 生态成熟、文档完善 |
| **Tushare Pro（200-500 元/年）** | 生产级数据获取 | 数据质量高、接口稳定、社区庞大（30 万+用户） |
| **QMT（券商开通）** | 实盘交易 + Tick 行情 | 免费、直连交易所、支持自动交易 |

#### 第二梯队（补充选择）

| 渠道 | 使用场景 |
|------|---------|
| **Baostock** | 需要稳定免费 A 股历史数据，不想折腾反爬 |
| **聚宽 JQData** | 需要清洗好的数据做回测，不着急实盘 |
| **iTick 免费版** | 需要毫秒级实时行情，盯盘/预警场景 |

#### 为什么是这个排序？

1. **成本优先**：个人开发者对价格极度敏感，免费/低价方案优先
2. **Python 生态**：AKShare / Tushare / 聚宽均为 Python 原生，入门门槛低
3. **实盘刚需**：一旦走到实盘，QMT 是门槛最低的合规交易+行情通道
4. **社区效应**：Tushare 社区庞大，遇到问题容易找到解决方案

## 四、推荐方案

### 4.1 盯盘 AI 助手场景推荐

针对**"A 股自动盯盘 AI 助手"**这一产品形态，推荐以下分层架构：

#### 主数据通道（推荐）：Tushare Pro + QMT 组合

```
┌─────────────────────────────────────────────┐
│  实时盯盘/预警  →  QMT（券商接口）           │  ← 毫秒级 Tick/五档
│  策略研究/回测  →  Tushare Pro（500元/年）   │  ← 稳定日线/分钟线
│  备份/应急      →  AKShare / Baostock        │  ← 免费兜底
└─────────────────────────────────────────────┘
```

| 用途 | 推荐渠道 | 理由 |
|------|---------|------|
| **实时行情（盯盘）** | QMT 或 iTick 免费版 | 延迟最低，适合实时预警 |
| **历史数据（回测）** | Tushare Pro 5,000 积分档 | 500 元/年，数据质量稳定，分钟线可另购 |
| **数据备份/冗余** | AKShare / Baostock | 免费，主通道故障时应急 |
| **财务/基本面数据** | Tushare Pro | 财务数据标准化程度高 |

#### 备用切换方案

| 场景 | 主方案 | 备用方案 | 切换条件 |
|------|--------|---------|---------|
| 实时行情中断 | QMT | iTick 免费版 / 聚宽 | QMT 券商系统维护时 |
| 历史数据接口故障 | Tushare Pro | Baostock / AKShare | Tushare 服务异常时 |
| 分钟线需求 | Tushare 付费分钟包 | 掘金专业版 | 需要更长历史分钟线时 |
| 全量市场扫描 | Tushare 实时日线 | AKShare `stock_zh_a_spot_em` | 需全市场快照时 |

### 4.2 纯个人零成本方案

若预算严格为零：

```
┌─────────────────────────────────────────────┐
│  实时全市场快照  →  AKShare（带限流+缓存）   │
│  历史数据        →  Baostock                │
│  基本面数据      →  Tushare 免费版（120积分）│
└─────────────────────────────────────────────┘
```

> ⚠️ 零成本方案**不适合高频盯盘**，AKShare 实时性不足且不稳定，仅适合日内低频扫描或收盘后分析。

## 五、风险点

### 5.1 数据合规风险矩阵

| 风险等级 | 渠道 | 具体风险 |
|---------|------|---------|
| 🔴 **极高** | 爬虫（东方财富/雪球） | 反不正当竞争法、非法获取计算机信息系统数据罪 |
| 🟡 **中高** | AKShare（商用） | 底层爬虫无授权，商用可能被追诉 |
| 🟡 **中** | Tushare Pro | 商用边界模糊，"不可转售或对外提供数据服务" |
| 🟢 **低** | QMT / iTick / Wind / iFinD | 正规授权通道，合规 |

### 5.2 稳定性风险

| 渠道 | 风险描述 |
|------|---------|
| AKShare | 东方财富反爬升级是**不可逆趋势**，接口随时失效 |
| Tushare 免费版 | 限频极低（50 次/分），无法满足全市场扫描 |
| 交易所官方 | 个人无法直接接入，需通过持牌供应商中转 |

### 5.3 成本陷阱

- **Tushare 的"隐藏成本"**：积分只是基础门槛，分钟数据、实时数据、港美股均需**额外单独付费**
- **QMT 的"隐性门槛"**：虽免费，但需维持证券账户资金量（10-50 万），且不同券商佣金差异大
- **Wind/iFinD 的"报价游戏"**：公开报价数万元/年，实际机构采购可谈至 5 折以下，但个人无法议价

### 5.4 实时性风险

| 渠道 | 典型延迟 | 是否适合实时盯盘 |
|------|---------|----------------|
| AKShare | 15-30 分钟 | ❌ 否 |
| Tushare Pro | 200-500ms | ⚠️ 勉强可用，高峰期拥堵 |
| iTick | <100ms | ✅ 是 |
| QMT | 毫秒级 | ✅ 是 |
| 交易所官方 | 最低 | ✅ 是（但个人无法接入） |
